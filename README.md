# Combo Planner

Combo Planner は、ユーザーが自分で登録したキャラクター、PT、技種別、コンボルートをローカルで管理・表示する Windows 向けの非公式ツールです。

本ツールは、特定のゲーム会社、運営会社、権利者とは関係ありません。  
配布ZIPには、ゲーム画像、公式データ、利用者データ、攻略サイト由来データは含まれていません。

## 対応環境

- Windows 10 / 11 x64
- Microsoft Visual C++ Redistributable x64 が必要です

## 使い方

ZIPを展開し、`ComboPlanner.exe` を起動してください。  
`ComboPlanner.exe` だけを別の場所へ移動せず、展開したフォルダ構成のまま使用してください。

初回起動時に、アプリと同じフォルダ内に `data/` フォルダが作成されます。  
登録したキャラクター、PT、コンボルート、画像などはローカルの `data/` に保存されます。

更新や削除の前には、必要に応じて `data/` をバックアップしてください。
